# India-s-Got-Latent-DataEngineering-Project
# LatentAnalytics

## Objective

Analyze audience engagement and sentiment for India's Got Latent Season 2 Episode 1.

## Dataset

- YouTube Data API v3
- 53.2M Views
- 2.56M Likes
- 100K Comments

## Features

- API Integration
- Comment Scraping
- Text Cleaning
- Sentiment Analysis
- Keyword Analysis
- Word Cloud
- Streamlit Dashboard

## Results

67% Neutral
25% Positive
8% Negative

## Technologies

Python
Pandas
TextBlob
Matplotlib
WordCloud
Streamlit
YouTube API
